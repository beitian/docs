Emacs
===

Hotkeys
---

以下是emacs中的一些快捷键

### 列编辑

 操作 | 快捷键                                         
------|------------------------------------------------
 替换 | C-x r t string（如果string为空格，则插入空格） 
 删除 | C-x r d/k                                      
 复制 | C-x r y                                        
      |                                                

- C-x r k 剪切一个矩形块
- C-x r y 粘贴一个矩形块
- C-x r o 插入一个矩形块
- C-x r c 清除一个矩形块(使其变成空白)
- C-x r t 在选定区域的所有列前插入样的字符
